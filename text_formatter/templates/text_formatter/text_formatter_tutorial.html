Text Formatter Tutorial

Salin dari WhatsApp Web

Untuk menyalin teks dari WhatsApp Web, Anda dapat menggunakan fitur salin teks yang ada di WhatsApp Web. Untuk memastikan format teks (bold, italic, dll.) ikut tersalin, gunakan perintah salin yang berada pada kanan atas dari teks yang ingin disalin.

Tempelkan ke Text Formatter

Setelah menyalin teks dari WhatsApp Web, pergi ke Text Formatter dan tempelkan pada input untuk WhatsApp.

Setelah menempelkan teks pada input untuk WhatsApp, hasil format akan muncul pada output untuk Telegram dan Web Narasi. Klik tombol Copy dan tempelkan pada Telegram atau Web Narasi.
